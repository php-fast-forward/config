FastForward Config
==================

FastForward Config helps PHP applications build one predictable configuration object from arrays, PHP files, and provider classes. It is designed to stay easy for first-time users while still supporting lazy loading, PSR-11 container integration, and PSR-16 caching for production setups.

If you are new to the package, start with :doc:`getting-started/quickstart`. If you already know the kind of source you want to load, jump to :doc:`usage/getting-services`.
